URL Shortener Microservice

This is a url shortener microservice written with Go as a solution to Free Code Camp's URL Shortener Microservice API Project

View Source

Instructions

  • Create a new shortened url {{ .Host }}/new/http://google.com Expected output: 
    {
    original_url: "http://google.com",
    short_url: "{{ .Host }}/px4OAI11"
}
  • Example Usage {{ .Host }}/px4OAI11 Will redirect to: http://google.com
